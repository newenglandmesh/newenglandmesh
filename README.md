# New England Mesh website

This repository powers the New England Mesh website at
`https://newenglandme.sh/`.

The site promotes MeshCore and Meshtastic activity across New England, links to
regional tools, and collects practical docs for getting on the air and operating
shared infrastructure.

## Regional links

- [New England Mesh map](https://map.newenglandme.sh/)
- [New England Mesh analyzer](https://analyzer.newenglandme.sh/)
- [New England Mesh healthcheck](https://healthcheck.newenglandme.sh/)
- [CT Mesh](https://ctmesh.org/)
