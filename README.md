# New England Mesh

This repository powers the New England Mesh website and documentation. The
project supports MeshCore and Meshtastic activity across New England, connects
operators with their local communities, and documents practical ways to build
and operate resilient mesh infrastructure.

## New England Mesh-hosted services

- [Main website and documentation](https://newenglandme.sh/)
- [MeshCore live map](https://map.newenglandme.sh/)
- [MeshCore analyzer](https://analyzer.newenglandme.sh/)
- [MeshCore region boundaries](https://newenglandme.sh/regions/)
- [Service healthcheck](https://healthcheck.newenglandme.sh/)
- [New England Mesh Discord](https://discord.newenglandme.sh/)

The New England Mesh map and analyzer combine regional data for Connecticut,
Massachusetts, and New Hampshire. MeshCore observers should upload through
their local community's MQTT service rather than directly to the regional
backend.

## Regional communities
