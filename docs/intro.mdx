# Get started

New England Mesh helps people in the region build, test, and document resilient
LoRa mesh communications. The focus is practical: get a node on the air, find
nearby operators, improve coverage, and share what works.

## What to build

- **MeshCore** is a newer LoRa mesh platform with personal Companion devices
  plus infrastructure roles such as Repeater and Room Server.
- **Meshtastic** is a mature, popular LoRa mesh platform with mobile apps,
  hardware guides, public maps, MQTT support, and a large community.
- **Local infrastructure** includes well-placed Repeaters, Room Servers,
  Routers, MQTT integrations, public maps, and documentation that helps
  neighbors discover the mesh.

## First steps

1. Pick a supported 915 MHz LoRa device for the platform you want to try.
2. Flash the firmware with the official web flasher or project instructions.
3. Pair the node with the mobile or web app.
4. Use conservative public settings until you know the local channel plan.
5. Test range from home, a hill, or another legal operating location.
6. Share coverage notes, useful locations, and corrections with the community.
